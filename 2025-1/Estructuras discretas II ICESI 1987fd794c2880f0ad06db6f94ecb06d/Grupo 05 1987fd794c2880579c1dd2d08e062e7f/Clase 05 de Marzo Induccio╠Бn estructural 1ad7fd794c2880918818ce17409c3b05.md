# Inducción estructural

Listas

- Lista en su paso recursivo: <cabeza> <cola> donde la <cola> es una lista
- Caso base es la lista vacia nil o List()

# Ejemplo Desbordamiento

```cpp
#include <iostream>

using namespace std;

int main() {
  // Entero maximo
  int a = 2147483647;
  cout << a << endl;
  cout << a + 1 << endl;
}

```
