# Tensorflow

Que es tensorflow

Es una librería para el procesamiento de datos que trabaja directamente con las instrucciones del PROCESADOR (GPU), por ello es conveniente tener esta aceleración

---

Que es un tensor

Son vectores de n dimensiones que están optimizados para operaciones algebraicas, es la unidad básica de tensorflow para representar información, estan optimizados para operaciones y uso de memoria

---

¿Que podemos hacer con tensorflow?

1. Cargar datos
2. Aplicar una función a un conjunto de datos
3. Filtrar datos
4. Reorganizar datos
5. En general ETL
